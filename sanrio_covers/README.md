# sanrio_covers — キャラクター画像の置き場

`sanrio_movie.html` が読み込むキャラクター画像をここに置きます。

## 命名規則
ファイル名の番号 = **順位**。`sanrio_data.csv` の「画像」列と対応します。

| ファイル | 順位 | キャラ |
|---|---|---|
| c1.png | 1位 | ポムポムプリン |
| c2.png | 2位 | シナモロール |
| c3.png | 3位 | ポチャッコ |
| … | … | … |
| c20.png | 20位 | コロコロクリリン |

- 推奨アスペクト比 **3:4**（縦長カード）。高解像度ほどKen Burns/全面化に有利。
- `.png` 以外を使う場合は `sanrio_data.csv` の「画像」列の拡張子も合わせて変更し、`python3 build_sanrio_movie.py` で再ビルド。
- **画像が無い場合**は自動でパステルの色付きカード（キャラ名入り）にフォールバックします。シルエット演出（伏せ→公開）はカードでも成立します。

# sounds — 効果音(SE) / BGM の差し替え置き場

`sanrio_movie.html` の SEレイヤーは **Web Audio で合成した音**で鳴ります（外部ファイル不要）。
このフォルダに **同名の mp3** を置くと、その音に**自動で差し替わり**ます。

| ファイル名 | 鳴るタイミング |
|---|---|
| `pop.mp3`     | クイズのヒントが1つ出るたび |
| `ding.mp3`    | キャラ正解の公開（リビール）瞬間 |
| `sparkle.mp3` | 公開と同時のキラキラ |
| `fanfare.mp3` | TOP10 / BEST3 / No.1 バナー・ED |
| `whoosh.mp3`  | スライド切り替えのスワイプ |
| `tick.mp3`    | 投票数カウントアップ中の刻み（短め推奨） |
| `heart.mp3`   | カウント中のドキドキ（短め推奨） |

BGM は親フォルダの `movie_bgm.mp3` を置くと再生されます（画面下「🔇 音OFF」を押すと ON）。

## フリー素材のおすすめ（商用OK・クレジット不要が中心。利用前に各サイトの規約を確認）
- **効果音ラボ** … 日本語UI・SEが豊富
- **DOVA-SYNDROME** … BGM
- **甘茶の音楽工房** … かわいい系BGM
- **Pixabay / Mixkit** … SE・BGM（ロイヤリティフリー）

※ クレジット表示が必要な素材（一部のCC等）は、概要欄等に明記してください。
